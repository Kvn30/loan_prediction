# Loan Prediction System

A Machine Learning based web application that predicts whether a loan application is likely to be approved or rejected based on applicant details.

## 📌 Project Overview

The Loan Prediction System helps financial institutions automate the loan approval process using Machine Learning algorithms. The system analyzes applicant information such as income, education, employment status, credit history, and loan amount to predict loan approval status.

This project is built using Python, Scikit-learn, Pandas, and Streamlit.

---

## 🚀 Features

* Predict loan approval status instantly
* User-friendly web interface using Streamlit
* Data preprocessing and feature encoding

## 📊 Dataset Information

The dataset contains applicant details such as:

* Gender
* Marital Status
* Education
* Self Employed
* Applicant Income
* Coapplicant Income
* Loan Amount
* Loan Amount Term
* Credit History
* Property Area
* Loan Status

Dataset Source: Kaggle Loan Prediction Dataset.
